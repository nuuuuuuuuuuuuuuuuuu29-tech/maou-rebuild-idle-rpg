# 手動QA手順

この手順は、非エンジニアでも「魔王再建記」が壊れていないか確認できるようにするためのものです。問題が出た場合は、最後のテンプレートをCodexへ貼ってください。

## 1. 起動確認

- [ ] PowerShellまたはターミナルでプロジェクトフォルダを開く
- [ ] `npm.cmd install` を実行する
- [ ] `npm.cmd run dev` を実行する
- [ ] 表示されたURLをブラウザで開く
- [ ] ホーム画面に「魔王再建記」が表示される
- [ ] ブラウザの開発者ツールで console error が出ていない

## 2. 初回プレイ確認

- [ ] 初回案内が表示される
- [ ] 魔王名を変更できる
- [ ] ホーム画面の「次にやること」が読める
- [ ] 遠征準備画面へ移動できる
- [ ] 初回おすすめダンジョンが分かる

## 3. 遠征確認

- [ ] 「煤けた境界村」を選ぶ
- [ ] 待機中の魔物を1体以上選ぶ
- [ ] 作戦方針を選ぶ
- [ ] 危険度ラベルが表示される
- [ ] 遠征開始ボタンを押せる
- [ ] 遠征中の残り時間と進捗バーが表示される
- [ ] 遠征完了後、作戦記録へ結果が表示される

## 4. 報酬確認

- [ ] 遠征結果に金貨が表示される
- [ ] 魔王EXPが表示される
- [ ] 配下EXPが表示される
- [ ] 領地解放率が表示される
- [ ] アイテムまたは救出魔物がある場合、報酬欄に表示される
- [ ] MVP表示が自然に読める

## 5. 成長確認

- [ ] 遠征後に魔王Lvまたは魔王EXPが変化している
- [ ] 配下一覧でユニットEXPまたはLvが変化している
- [ ] 戦闘不能になったユニットがいる場合、状態が分かる
- [ ] 詰まった場合でも司令部で基本ユニットを雇用できる

## 6. 図鑑確認

- [ ] 図鑑画面へ移動できる
- [ ] 発見済みの魔物、アイテム、ダンジョンが表示される
- [ ] 未発見のものが「？？？」で表示される
- [ ] アイテムのレアリティが表示される

## 7. 実績確認

- [ ] 実績欄が表示される
- [ ] 達成済み実績が完了状態で表示される
- [ ] 未達成実績の進捗が表示される
- [ ] 遠征完了時の実績解除ログが読める

## 8. 称号確認

- [ ] 図鑑画面で称号一覧が表示される
- [ ] 獲得済み称号を選択できる
- [ ] 未獲得称号は選択できない
- [ ] ホーム画面に選択中称号が表示される
- [ ] リロード後も選択中称号が維持される

## 9. 熟練度確認

- [ ] 遠征準備画面でダンジョン熟練度が表示される
- [ ] 図鑑のダンジョン欄で熟練度と踏破回数が表示される
- [ ] 成功したダンジョンの踏破回数が増える
- [ ] 失敗や撤退では踏破回数が増えない

## 10. レアドロップ確認

- [ ] 遠征準備画面で希少候補が表示されるダンジョンがある
- [ ] Rare以上のアイテムを入手した場合、報酬欄で目立つ
- [ ] 初入手のRare以上アイテムが図鑑に登録される
- [ ] インベントリ満杯時に未受取アイテムが勝手に図鑑登録されない

## 11. 魔物特性確認

- [ ] 配下一覧で魔物特性が表示される
- [ ] ユニット詳細で特性説明が読める
- [ ] 遠征準備画面で部隊特性ボーナスが表示される
- [ ] リロード後も既存ユニットに特性が表示される

## 12. 危険度UI確認

- [ ] 初期部隊で「煤けた境界村」を選ぶと `安全` または `注意` になる
- [ ] 初期部隊で「黒玻璃の林」を選ぶと `危険` または `無謀` になる
- [ ] `強行突破` や `戦利品重視` にすると危険度が下がりすぎない
- [ ] 危険度理由が最大2件程度で読みやすい
- [ ] おすすめ行動が1行で表示される
- [ ] `無謀` でも出撃ボタンは押せる

## 13. 保存/復元確認

- [ ] 魔王名を変更する
- [ ] 遠征を1回完了する
- [ ] 称号を選ぶ
- [ ] ブラウザをリロードする
- [ ] 魔王名、所持金、配下、アイテム、図鑑、称号が維持されている
- [ ] 遠征中にリロードしても、終了時刻後に結果が反映される

## 14. スマホ幅確認

- [ ] ブラウザ幅を375px程度にする
- [ ] ホーム画面が横スクロールしない
- [ ] 遠征準備画面が横スクロールしない
- [ ] 危険度パネルの文章が読める
- [ ] 作戦記録の報酬表示が読める
- [ ] 図鑑、実績、称号が大きく崩れない
- [ ] ナビゲーションが押せる

## 15. リセット確認

- [ ] 設定画面へ移動する
- [ ] リセットボタンを押す
- [ ] 確認ダイアログが表示される
- [ ] キャンセルするとデータが残る
- [ ] 実行すると初期状態に戻る
- [ ] バックアップに関するメッセージが表示される
