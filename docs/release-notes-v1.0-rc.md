# 魔王再建記 v1.0 RC リリースノート

この文書は、v1.0公開候補として現在プレイヤーが使える主な機能をまとめたものです。開発履歴ではなく、遊ぶ側から見える内容を中心にしています。

## 概要

「魔王再建記」は、敗北した魔王が配下の魔物を集め、遠征で資源と領地を取り戻していく放置型・遠征型・魔物育成RPGです。

フロントエンドのみで動作し、セーブデータはブラウザのlocalStorageに保存されます。画像素材や外部APIは使わず、絵文字、テキスト、CSSでダークファンタジー調の雰囲気を作っています。

## プレイヤーができること

- 魔王名を設定する
- 魔物を雇用、救出、育成する
- 複数の魔物で遠征部隊を編成する
- ダンジョン、作戦方針、持ち込みアイテムを選んで遠征する
- 遠征ログ、戦闘結果、報酬、MVPを確認する
- 所持金、経験値、アイテム、救出魔物、領地解放率を増やす
- 司令部で雇用、購入、売却、保有上限拡張を行う
- 図鑑、実績、称号、熟練度、ボス討伐記録を集める

## 主な収集・やりこみ要素

### 実績

遠征、勝利、ボス討伐、図鑑登録、魔王Lvなどに応じて実績が解除されます。解除された実績は図鑑画面と作戦ログで確認できます。

### 図鑑報酬

魔物、アイテム、ダンジョン、総登録数が一定数に到達すると、図鑑画面から報酬を受け取れます。受け取りは一度だけで、アイテム枠不足時はブロックされます。

### ボス討伐記録

ダンジョンごとにボス討伐回数、初討伐日、最終討伐日が記録されます。

### ダンジョン熟練度

同じダンジョンを成功クリアすると踏破回数が増え、熟練度Lvが上がります。熟練度は獲得金額とユニット経験値に控えめなボーナスを与えます。

### レアドロップ

Rare、Epic、Legendaryのアイテムは希少品として扱われます。実際にインベントリへ入った希少品だけが、図鑑、実績、ログへ反映されます。

### 称号

進行状況から称号を獲得し、ホーム画面に掲げられます。称号はステータス効果を持たない、達成感と表示用の要素です。

### 魔物特性

魔物ごとに種族固定の特性があります。遠征前に部隊特性ボーナスを確認し、編成を少し考える楽しさを作っています。

### 危険度UI

遠征準備画面で、現在の部隊、ダンジョン、作戦方針から危険度を `安全` / `注意` / `危険` / `無謀` の4段階で表示します。成功率の具体的な%は表示せず、出撃も禁止しません。

## セーブと互換性

- 現在の `GameState.version` は `5` です。
- localStorageキーは `maou-rebuild-state-v1` です。
- v2、v3、v4相当の古いセーブデータから現行形式へ移行できます。
- 移行前のデータはバックアップキーへ退避します。
- 壊れたセーブデータを検知した場合も、可能な限りバックアップしてから初期化します。

## 確認済みの自動検証

- 遠征開始と遠征完了
- 報酬、経験値、所持金、図鑑反映
- 実績解除
- ボス討伐記録
- 図鑑報酬の受け取り、二重受け取り防止、枠不足ブロック
- ダンジョン熟練度
- レアドロップ
- 称号判定と選択
- 魔物特性
- 遠征危険度判定
- localStorage保存/復元
- 古いセーブデータ移行
- 壊れたセーブデータ復旧
- バランスシミュレーションの再現性

## 既知の制限

- セーブデータはブラウザlocalStorageに保存されます。別ブラウザや別端末には自動同期されません。
- ブラウザのサイトデータを削除するとセーブデータも消えます。
- E2Eテストは未導入です。公開前は `docs/manual-qa.md` に沿って手動確認してください。
- 装備、合成、転生、複数部隊同時遠征、本格的な長時間放置モードは未実装です。
